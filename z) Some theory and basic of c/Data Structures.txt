Data are simply values or set of values. A data item
refers to a single unit of values.
An entity is something that has certain attributes or
properties which may be assigned values.
Attributes: Name  Age
Values:     Moloy 23

Information is sometimes used for data with given
attributes, or in oter word meaningful or processed
data.

Data Structures: The logical or mathematical model
of a particular organization of data is called a
data structure.
Primiitive: Basic data types. These data types consist
of characters that cannot be divided and hence they are
also called simple data type.
A data structure is said to be linear if its elements
from a sequence or a linear list.
A data structure is said to be non-linear if the data
is not arranged in sequence.
